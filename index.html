1.1 群组概况

OverdoseWiki群组重建于2025年9月，至2026年5月7日数据采集截止，共产生74,434条消息，来自943名发言用户。群组活跃天数224天，日均消息332条，最活跃的一天（2026年4月26日）产生了3,394条消息，最安静的一天仅有11条。

1.2 活跃时段：深夜是主场

74,434条消息的发送时间分布呈现出鲜明的昼夜差异。0-5点时段合计25,659条，占全天消息量的34.5%，而6-11点时段仅有9,141条，占12.3%。凌晨1点是全天最活跃的单小时，产生5,941条消息；上午8点则是最低谷，仅1,040条。

1.3 增长轨迹：4月爆发

群组的消息量并非匀速增长。2025年9月群重建首月仅有1,111条消息，10月迅速攀升至12,742条，此后11月至3月在5,250-8,344条之间波动。2026年4月，消息量突然飙升至16,824条，为观测期内的峰值，占全量消息的22.6%。

4月的爆发集中在几个关键日期：4月10日（1,002条）、16日（1,044条）、26日（3,394条）、27日（2,447条）、30日（1,228条）。其中4月26-27日的爆发尤为显著——两天合计5,841条消息，占全月消息量的34.7%。

这一时间段的爆发与外部事件相关。4月2日，深圳市公安局、卫健委和市场监管局联合发布通告，将含右美沙芬复方制剂列入禁毒监控对象清单；4月8日，广州禁毒官方视频引用了群组关联的Wiki内容。管控收紧的信号反而激发了群组内的密集讨论——成员们讨论如何囤药、如何寻找替代渠道、以及如何应对可能的执法行动。4月26日当天，群内充斥着"美和pr没了""现在哪里可以买白兔""京东碰运气"之类的消息。

2.2 剂量分享：数字化的风险传递

剂量分享是群组内最核心的药物相关话题之一，共2,206条消息，占比2.96%。其中含具体剂量数值的消息涉及1,548个剂量数据点。剂量分布呈现出明显的向高剂量偏移的特征：

图5 · 剂量分布图——向高剂量偏移的特征

在群组的暗语中，"t"代表"片"（tablet）。以右美沙芬为例，一粒愈美片含右美沙芬15mg，正常治疗剂量为每次1-2片，最大日剂量不超过8片（120mg）。10t意味着一次性服用10片（150mg），已超出最大日剂量；20t（300mg）是最大日剂量的2.5倍；而50t（750mg）则是6倍以上。群内出现的极端案例包括"770t右美"（11,550mg，正常日剂量的96倍）、"600t套"和"200t茶苯海明"。

1,548个剂量数据的平均值为24.1t。这意味着，平均每次分享的剂量已经远超正常治疗范围。剂量分享消息的共现标签揭示了其功能特征：23.5%同时带有"药物名称提及"——分享者通常会说明具体药物品种；15.9%带有"用药方式讨论"——剂量与用法往往是配套传递的信息；12.0%带有"经验分享"——剂量分享本质上是个人用药经验的公开化；4.9%带有"煽动美化"——高剂量分享往往伴随着对药物体验的正面描述。

这种"剂量-药物-用法-体验"的信息链条，将个体用药行为转化为可复制、可传播的知识，降低了后来者的尝试门槛。

这类用户以分享药物知识和用药经验为主要行为。他们的表达方式往往带有"科普"色彩——"我来整理一下右美的正确用法，首先剂量要控制在……"——这种姿态使药物使用看起来像一门可以系统学习的"知识"。代表用户中，"虚假的 溺安"（3,764条发言，经验分享99条、药效讨论203条）和"LuoYunXi"（954条发言，经验分享63条、药效讨论64条）是典型。前者同时也是群组中煽动美化标签最密集的用户之一（70条），说明"知识分享"与"价值引导"往往发生在同一个人身上。

交易撮合型

这类用户主要参与药品交易和购药渠道的信息流通。"LuoYunXi"同时出现在知识权威和交易撮合两个类型中——处方购药渠道标签101条，药品交易32条。另如"偷吃尸体的 老鼠"（1,117条发言，处方渠道65条、药品交易13条）和"msh usd"（516条发言，药品交易20条、处方渠道30条），他们的发言频繁涉及购药渠道和价格信息。担保/代收相关消息22条，意味着群内存在某种非正式的"信用机制"——交易撮合者有时充当中间人角色。

煽动美化型

"夜 夜"（1,271条发言，煽动美化76条）是群组中煽动美化标签最密集的用户。"fly bug"（1,377条发言，煽动美化52条）和"u pucs"（1,115条发言，煽动美化49条）紧随其后。群内"化学极乐"（31次）、"飞升"（18次）、"灵魂出窍"（10次）等美化表达，部分正是由这类用户反复使用和传播的。煽动美化标签月度趋势显示，2026年4月达到峰值509条，5月进一步攀升至436条（仅7天数据）。这与4月管控收紧后的群体应激反应一致——外部压力越大，内部的话语体系就越倾向于强化对药物使用的正面描述。

3.4 权力结构：隐形的群主与缺席的治理

群主OverdoseWiki在224天内仅发言44条，占比0.06%。这些发言大多是功能性消息——"帮我把之前的人都拉回来""不如你频道把重新加群链接挂上"——而非对群组内容的管理或引导。群主偶尔也参与药物讨论："戒断反应""熬过去就好了，或者苯海索/劳拉西泮，少吃一点就ok"——群主在建议如何应对戒断反应时，给出的方案仍然是"吃药"。

图11 · 权力结构对比——名义权力 vs 实际影响力

群组内与"管理"相关的消息极少："规则"仅12条，"欢迎"12条，"管理"43条（多为成员讨论权限问题），"封禁"3条。没有成文的群规，没有内容审核，没有行为约束——OverdoseWiki处于事实上的无治理状态。

群组内唯一的Bot（kmuav2bot）提供的是娱乐功能，而非安全功能。它的主要交互是娱乐性质的——扔瓶子、捡瓶子、waifu、涩图。这与群组的定位一致：药物过量在这里不是需要被阻止的危险，而是被分享和传播的内容。既然如此，安全功能自然没有必要存在。

这种治理真空并非偶然——它反映了群组的核心逻辑：信息自由流通、行为不受约束。当有人发出"救命"时没有管理机制介入，当有人分享极端剂量时没有内容被移除，当未成年人出现在群组中时没有身份验证或劝退。群组作为一个社会空间，其"规则"完全由核心用户的发言模式和话语体系自发形成。

3.5 去中心化网络

OverdoseWiki并非一个孤立群组。通过分析消息中引用的外部链接和频道，可以勾勒出一个跨群组的传播网络。

图12 · 外部网站引用频次

bilibili.com和抖音的出现尤其值得关注——这两个以年轻用户为主的平台，正在成为药物相关信息传播的中转站。bilibili上的视频内容可能以"科普"或"体验分享"的形式存在，经过算法推荐触达更广泛的年轻受众。

这个网络的结构特征是去中心化——没有单一的中心节点，各个频道和平台之间通过链接和转发形成松散的连接。即使OverdoseWiki群组被封禁（群组确实经历过多次封禁后重建），信息流通可以通过其他节点继续维持。封禁只是暂时移除了一个入口，而网络本身的结构并未被破坏。

黎明前的黑暗——最深的夜，也是最接近光的时候

这个群组存在的这八个月，不是凭空出现的，是在一个更大的背景下生长出来的。

一、滚动的列管清单，和它追不上的速度

过去几年，监管部门发挥了它该有的效果。2021年12月，国家药监局将右美沙芬口服单方制剂由非处方药转换为处方药；2022年11月，药品网络销售禁止清单（第一版）出台，右美沙芬单方制剂被列入禁止网络零售名单。2023年2月，国家药监局、公安部、国家邮政局等三部门联合下发通知，明确我国部分地区出现右美沙芬等五类药品的滥用问题，要求严格控制药品生产量，强化经营环节监管，加强寄递渠道查验。2024年4月，国家药监局、公安部、国家卫健委联合发布公告，自2024年7月1日起，将右美沙芬正式列入第二类精神药品目录。

听上去，这条路正在被堵上，但群组的聊天记录告诉我们它没有。右美沙芬被列管后，部分滥用者迅速转向愈美片等复方制剂获取药物——而愈美片不在列管范围内，不需要处方，八块钱一盒，仍可在各地药店随意购买。在群组里，"愈美"这个词从九月二十六日第一天起就频繁出现，从未消失过。

这就是这类治理困境的结构：管住了一种，旁边的门还开着。国内买不到，去日本；日本买不到，找美国代购。群组里的跨境购药讨论涉及三大洲、多条线路。监管的边界是地理性的，而网络不是。

二、知道问题的人，和问题本身的规模

《2024年中国毒情形势报告》显示，截至2024年底，中国现有吸毒人员74.7万名，同比下降16.7%，戒断三年未发现复吸人员428.6万名，同比上升5.1%。传统毒品的数字在向好的方向走。但同一份报告里，另一组数字的方向是反的。国内滥用物质种类发生结构性变化，麻精药品和未列管成瘾性物质滥用快速蔓延，滥用人数不断增多，青少年滥用问题突出。2024年查处35岁以下滥用麻精药品人数达7.6万人次，占查获滥用总人数的86.4%；全年共发现滥用未列管成瘾性物质人数4.9万人次。

这两条数字并列放在一起，意味着什么？传统毒品的流行正在被抑制，这是真实的成果；但另一批年轻人，正在通过药店和快递，以另一种路径滑入同样的处境。他们中的很多人，不觉得自己是"吸毒者"。他们买的是止咳药，是处方药，是网上搜得到的东西。这个认知距离，是治理的真正难点所在。

三、那些还没人管的地方

这个群组里最难被任何法规触碰到的，不是剂量，不是购药渠道，而是两件事：

第一，是那个叫od-wiki.com的网站，和它背后的知识生产逻辑。用科学语言包装的用药指南，不是在贩卖毒品，不是在发布违禁信息——它只是在"分享知识"。现行法规对此几乎没有针对性条款。

第二，是群组里那943个人的共同处境。他们当中，有初中生，有高考前的少年，有凌晨两点崩溃发来"谁可以帮帮我"的人。青少年在青春发育期伴随焦虑、抑郁等情绪问题日益突出，而药物滥用往往成为他们寻求短暂解脱的一种方式。这句话在报告里只是一行背景说明，但在这七万条消息里，它是具体的面孔、具体的凌晨时间戳、具体的那句"反正对外的正常生活维持不了了"。

他们进入这个群组，通常不是为了"嗑药"，而是为了找人说话。结果找到的，是一套完整的剂量指南。

四、现实的路怎么走

补上复方制剂的漏洞。右美沙芬单方制剂被列管之后，愈美片成了替代品，这个转移在几个月内就完成了。监管部门应当建立更快速的滥用预警—列管联动机制，不等问题规模化再响应。对于含有可滥用成分的复方制剂，应当明确每次销售限量，并对频繁、大量购买设置记录和预警机制。

认真对待网络销售的灰色地带。群组里的处方造假、电商平台的境外代购、快递寄递渠道，形成了一条完整的替代供应链。公安机关已破获利用外卖平台非法贩卖右美沙芬的案件，这是好的信号，但应当形成常态化的跨部门协作机制，而不是单次专项行动。

把"信息治理"纳入视野。仅靠封禁频道并不奏效——群组里置顶的那条公告已经证明了这一点：前管理员被警方调查，频道被封，但群组重建，继续运营。如何让平台对此类内容承担更明确的责任，包括自动检测危机信号并触发介入机制，是下一步的关键议题。

把青少年心理健康的缺口当成入口，而不是后果。这些孩子在加入药物群组之前，经历了什么？家庭的关爱、心理压力的疏导、平等的对话，是防范青少年走上这条路的关键——这话说起来容易，但学校心理健康服务的覆盖率、可及性和实际有效性，与这一表述之间还有很长的距离。更现实的问题是：一个初中生在凌晨两点崩溃，他有没有一条打得通、不需要实名、不用害怕被送去矫正学校的求助电话？

九月二十七日傍晚六点四十三分，有人在群里发了两个字："救命"。紧跟着的下一条消息，是另一个人问愈美片哪里买便宜。

"救命"被无缝接入了购药价格讨论。这两条消息之间，有某种我们应该感到不安的东西。不只是群组的问题，是一个在凌晨崩溃的孩子往外看，能看到什么的问题。他现在看到的，是这个群组。这不是因为他不想活，而是因为除了这个群组，他不知道还有什么地方可以发出声音，可以不被挂掉电话，不被送进矫正学校，不被当成一个需要被管住的问题。

224天的观察只是一个横截面。群组仍在运行，消息仍在产生。而在深夜的某个时刻，又一条"刚吃"的消息正在发送。
